Dra D se plnge c maina achiziionat cu ajutorul  celor de la Close  Motor Finance  Limited  
(CMF) nu este de calitate satisfctoare.
 
istoric
Dra D a cumprat o main uzat cu ajutor financiar n mai 2018. Maina avea 7 ani i 
jumtate la punctul de furnizare i strbtuse 85.738 mile. Maina cost 18.990  lire sterline.
Dra D a spus, la scurt  timp de la achiziia mainii, c maina avea probleme i avea nevoie 
de reparaii. Maina  a fost returnat la reprezentan a care a livrat-o i a fost reparat. Dra D 
a spus o lun mai trziu  c bateria trebuie s fie nlocuit. Ulterior, maina s-a stricat n 
momentul n care dra D conducea  n vacan i se pare c o alt reparaie a fost realizat de 
un garaj ter, ns maina avea n continuare probleme. S-a convenit  s se returneze  maina 
livratoare n vederea inspeciei deoarece dra D dorea s resping maina. Reprezentena 
care a livrat  maina a spus c maina a fost alimentat cu combustibil greit i nu era o 
problem la punctul  de vnzare. CMF a stabilit o inspecie independent a combustibilului,  
iar raportul  a concluzionat faptul c maina prezenta combustibil contaminat  n sistemul  
combustibil, i e posibil ca acest  lucru s fi provocat probleme cu motorul. CMF a spus c n 
aceste circumstane  nu sunt dovezi c maina era defect sau nu era de calitate 
satisfctoare la punctul de vnzare.  CMF a rambusat totui cinci trane  lunare i impozitul 
i asigurarea pentru a acoperi inconvenientul  provocat  de reparaii. Dra D nu credea  c 
acest lucru era suficient  i i-a naintat  reclamaia acestui  serviciu.
Anchetatorul nostru a luat n considerare reclamaia i nu a recomandat meninerea  sa. Ea a 
considerat c nu existau suficiente  dovezi  care s arate c maina avea vreo problem la 
punctul de furnizare.  
Dra D nu a fost de acord cu acest lucru i a rspuns spunnd  pe scurt c maina prezenta 
de la bun nceput numeroase probleme,  iar soluionarea lor a durat mult timp. Ea a rugat un 
ombudsman s ia n considerare reclamaia.
    
constatrile mele
Am analizat toate  dovezile i argumentele disponibile pentru a decide ce este corect i 
rezonabil n circumstanele acestei reclamaii.
Nu am niciun dubiu c dra D prezenta o serie de probleme cu maina  n perioada  de timp n 
care a fost n posesia sa. mi dau seama  c maina s-a stricat i a stat n garaj pentru 
reparaii n cteva  rnduri.
Ce trebuie eu s decid  este dac maina prezenta o calitate satisfctoare  n momentul 
furnizrii sale ctre dra D. Dac un lucru este sau nu de calitate  satisfctoare depinde de o 
serie de lucruri precum vechimea i kilometrajul  mainii i preul cu care a fost cumprat. n 
cazul drei D, maina avea o vechime  de aproximativ apte ani i jumtate i parcurse o 
distan de aproximativ  85.738 mile. Maina cost 18.990 lire sterline.  Calitatea  
satisfctoare acoper de asemenea durabilitatea care nseamn  c componentele din 
main trebuie  s fie durabile i s dureze o perioad rezonabil de timp,  ns din nou acest  
lucru va depinde de vechimea mainii i de numrul de mile parcuri. Cu alte cuvinte, dac o 
persoan rezonabil s-ar atepta c exist probleme de aceast natur la o main de 
aceast vechime, cu un asemenea pre i numrul  de kilometri.

Problemele raportate iniial de dra D erau legate de baterie  i alte lucruri. Se pare c 
acestea au fost reparate,  ns a rmas o problem  cu filtrul de particule (DPF) i injectoarele 
care se pare c ar fi provocat defectarea mainii. Nu dein foarte multe informaii legate de 
reparaiile care au fost realizate la main nainte  de august 2018.  Dra D a spus  c aceste 
reparaii suplimentare care au fost realizate de un garaj ter parte, motorul  tremura din nou 
i nu pornea. Maina a fost returnat  la reprezentana care a livrat-o deoarece nu funciona.
Inspecia care a fost realizat la main de inginerul ter parte  concluzioneaz  faptul c 
combustibilul folosit n main este contaminat. O mostr din combustibil a fost testat i a 
dezvluit particule  mari de murdrie  i resturi i de asemenea un coninut ridicat de sulf. 
Inginerul a tras concluzie c e posibil ca acest  lucru s fi deteriorat  injectoarele  i motorul.
Se pare c problemele cu maina se datoriaz  unei probleme cu tipul de combustibil folosit. 
ntruct dra D a putut conduce 4000 mile din momentul  n care a intrat  n posesia mainii, nu 
pot afirma c combustibilul  contaminat era acolo la momentul vnzrii  mainii. De 
asemenea, nu pot afirma  c problemele cu maina se datoreaz unei oarecare erori 
inerente sau unui defect care era deja acolo sau care a evoluat la punctul de vnzare. Dei 
maina a prezentat unele defeciuni care erau prezente  la punctul de vnzare, se pare c 
ele au fost remediate  de reparaii, iar problema actual se datoreaz combustibilului  
contaminat. Inginerul concluzioneaz c acest lucru trebuie s se fi produs la ultima 
alimentare i de aceea nu pot afirma c dra D ar trebui s fie lsat s resping maina.
Cred c msurile luate  de CMF  de rambursare  a plilor lunare i impozitului i asigurrii  
pentru a reflecta inconvenientul cauzat de reparaii sunt suficiente  i nu pot cere ca ele s 
mai fie fcute.
decizia mea final
Decizia mea final este c nu susin aceast  reclamaie. n baza regulilor  Financial 
Ombudsman Service, trebuie s i cer drei D s accepte sau s resping decizia mea nainte 
de data de 24 septembrie 2020.
